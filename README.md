# My Portfolio

This is my personal portfolio, built to showcase my work, skills, and journey as an engineering student, content creator, and freelancer.

## About

I'm Ayush, an engineering student who also creates content and takes on freelance work. This portfolio brings together my projects, skills, and content, and gives people a way to get in touch with me for freelance opportunities or collabs.

## Features

- Clean, single-page design
- Project and work showcase section
- About me / skills section
- Contact section with review/testimonial support (Firebase-backed)

## Tech Stack

- HTML, CSS, JavaScript
- Firebase (Firestore + Auth) for storing and managing reviews
- Deployed on Vercel
